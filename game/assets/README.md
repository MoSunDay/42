# 游戏资源目录

本目录用于存放游戏所需的所有资源文件。

## 目录结构

```
assets/
├── images/         # 图片资源（精灵、瓦片、UI等）
├── fonts/          # 字体文件
└── sounds/         # 音效和音乐
```

## 使用说明

### 图片资源 (images/)

将游戏所需的图片文件放在此目录：

- `player.png` - 玩家角色精灵
- `tileset.png` - 地图瓦片集
- `enemy_*.png` - 敌人精灵
- `ui_*.png` - UI 元素

**推荐格式**: PNG（支持透明背景）
**推荐尺寸**: 16x16, 32x32, 或 64x64 像素

### 字体资源 (fonts/)

自定义字体文件（可选）：

- `.ttf` 或 `.otf` 格式
- 游戏默认使用 Love2D 内置字体

### 音效资源 (sounds/)

音效和背景音乐文件：

- `.ogg` 或 `.wav` 格式
- 建议使用 OGG 格式以减小文件大小

## 获取免费素材

请参考项目根目录的 `docs/ASSETS_GUIDE.md` 文件，其中包含：

- 免费素材网站推荐
- 下载和使用指南
- 许可证说明

## 注意事项

1. **文件命名**: 使用小写字母和下划线，避免空格和特殊字符
2. **文件大小**: 尽量优化图片大小，避免过大的文件
3. **许可证**: 确保使用的素材符合许可证要求
4. **版本控制**: 大型资源文件建议使用 Git LFS
