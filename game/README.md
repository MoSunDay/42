## 🚀 快速开始

### 安装LÖVE2D

#### macOS
```bash
brew install love
```

#### Windows
从 [love2d.org](https://love2d.org/) 下载安装包

#### Linux
```bash
sudo apt-get install love
```

### 运行游戏

```bash
cd game
love .
```

## 🎮 操作指南

### 探索模式

| 按键 | 功能 |
|------|------|
| `WASD` / `方向键` | 移动角色 |
| `M` | 打开/关闭统一菜单 |
| `Tab` | 打开/关闭全屏地图 |
| `Enter` | 开始聊天输入 |
| `ESC` | 关闭打开的UI |
| `鼠标左键` | 点击地图移动 |

### 战斗模式

| 按键 | 功能 |
|------|------|
| `↑↓` | 选择动作 |
| `←→` | 选择敌人 |
| `Enter` / `Space` | 确认选择 |
| `ESC` | 取消 |
| `鼠标左键` | 点击选择 |

### 聊天输入

| 按键 | 功能 |
|------|------|
| `Enter` | 发送消息 |
| `ESC` | 取消输入 |
| `Backspace` | 删除字符 |

## 📖 系统说明

### 1. 账号与角色系统

#### 账号管理
- 用户注册与登录
- 密码验证
- 每个账号最多3个角色

#### 角色创建
- 自定义角色名字
- 选择外观（发型、肤色、服装）
- 初始属性分配

### 2. 地图系统

#### 地图探索
- 俯视角2D地图
- 实时小地图（右上角）
- 全屏地图（Tab键）
- 点击地图自动寻路

#### 可用地图
- **新手村** (town_01) - 3200x2400
- **春之镇** (spring_town) - 2400x2000

### 3. 战斗系统

#### 战斗流程
1. 遇敌触发战斗（明雷系统）
2. 战斗开场动画（2秒）
3. 玩家回合（选择动作）
4. 执行动作（动画播放）
5. 敌人回合（AI行动）
6. 检查胜负，继续或结束

#### 战斗动作
- **Attack（攻击）** - 对敌人造成伤害
- **Defend（防御）** - 减少受到的伤害
- **Item（道具）** - 使用道具（待实现）
- **Escape（逃跑）** - 50%概率逃离战斗
- **Auto Battle（自动战斗）** - 开启自动战斗
- **Cancel Auto（取消自动）** - 取消自动战斗

#### 回合计时器
- 每回合90秒操作时间
- 超时自动执行防御
- 实时倒计时显示

### 4. 装备系统

#### 装备槽位
- **Weapon（武器）** - 增加攻击力
- **Helmet（头盔）** - 增加防御力
- **Armor（护甲）** - 增加防御力

#### 装备管理
- 在统一菜单的 Equipment 标签页查看
- 显示已装备物品
- 显示总属性加成

### 5. 组队系统

#### 组队功能
- 创建队伍
- 邀请成员（最多5人）
- 队长管理
- 成员信息显示

#### 组队UI
- 队伍名称
- 成员列表
- 队长标识（金色★）
- 成员头像、等级、HP条

### 6. 聊天系统

#### 聊天功能
- 发送消息（Enter键）
- 消息历史（最多50条）
- 气泡消息（角色头顶）
- 气泡持续3秒并淡出

#### 聊天窗口
- 位置：左下角
- 显示最近消息
- 输入框激活状态

### 7. 统一菜单系统

按 `M` 键打开统一菜单，包含4个标签页：

#### Equipment（装备）
- 显示装备槽位
- 显示已装备物品
- 显示总属性加成

#### Items（道具）
- 道具列表（待实现）
- 道具使用（待实现）

#### Party（组队）
- 队伍信息
- 成员列表
- 队长管理

#### Pet（宠物）
- 宠物管理（待实现）
- 宠物战斗（待实现）

## 🛠️ 开发

### 项目结构
```
game/
├── main.lua              # 游戏入口
├── account/              # 账号系统
├── map/                  # 地图系统
├── src/
│   ├── core/            # 核心系统
│   ├── entities/        # 游戏实体
│   ├── systems/         # 游戏系统
│   └── ui/              # UI组件
├── assets/              # 资源文件
└── docs/                # 文档
```

### 代码规范
- 单文件不超过400行
- 模块化设计
- 清晰的命名规范
- 适当的注释

### 添加新功能

参考 [开发指南](docs/DEVELOPMENT_GUIDE.md) 了解如何：
- 添加新地图
- 添加新敌人
- 添加新UI标签页
